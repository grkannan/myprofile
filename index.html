Project Name: Online Voting Website (Academic Project)

Description of project:

Online voting system is the concept of voting via website from home or work place. This project makes the lowest expenses of cost to conduct the manual or machinery election. This web application allows the voters to watch the voting results. The candidates are selected and posted by the Election Commission (Admin). The selected or posted profiles are ready to collect the vote. The candidates are also voters. The voters only can nominate for the election. The nomination process is conducted via offline and it will conducted by the regional election officer. In this voting system, there are 10 constituencies which contain 100 voters for each. There HTML, CSS, JavaScript and PHP for frontend development and MYSQL are used for backend database.
FRONT END TOOLS: HTML, CSS, JavaScript and PHP.
BACKEND TOOLS: MYSQL (XAMPP Server).
